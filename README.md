В процессе разработки проекта нейросетевые инструменты использовались как вспомогательный инструмент для консультаций по HTML, CSS и JavaScript.

С помощью нейросети выполнялись:

* поиск и объяснение технических решений;
* помощь в создании отдельных CSS-анимаций;
* помощь в реализации интерактивных элементов на JavaScript;
* консультации по адаптивной вёрстке;
* проверка и исправление отдельных фрагментов кода;
* рекомендации по структуре проекта и организации файлов.

Все решения анализировались, адаптировались под задачи проекта и дорабатывались вручную. Финальная структура сайта, дизайн, вёрстка страниц и интеграция всех компонентов были выполнены самостоятельно.
